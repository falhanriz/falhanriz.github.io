Latest Projects

Automatic Irrigation System

The Automatic Irrigation System is an IoT-based project that automates plant watering using an ESP32 microcontroller integrated with the Blynk platform for remote monitoring and control. It features automatic scheduling via RTC and manual operation through a button or the Blynk app, supported by a DHT11 sensor to monitor temperature for optimal watering efficiency.

Arduino IDE C/C++ ESP32 Blynk

Design and Implementation of an Ammonia, Temperature, and Humidity Monitoring System in a Closed-House Chicken Coop Based on the Internet of Things

The Wireless Environmental Monitoring System consists of three sensor nodes, a master unit, and a gateway. Each node uses an ESP32 to collect data from the MQ-135 air quality sensor and the DHT11 temperature and humidity sensor, then sends the data to the ESP32 on the master unit. The master displays the readings on an LCD, activates a buzzer when ammonia levels exceed safe limits, and transmits the data to the gateway using LoRa E32. The gateway, powered by an ESP32, sends the data to the internet and stores it in a database, allowing real-time monitoring through a web dashboard and mobile application.

Arduino IDE C/C++ ESP32 LoRa
